# Third-Party Providers — SkyMetron

SkyMetron pode interagir com serviços externos configurados pelo usuário.

## Providers possíveis

- GitHub;
- OpenAI;
- NVIDIA;
- Google;
- Groq;
- Mistral;
- OpenRouter;
- Cerebras;
- MiniMax;
- Ollama/local.

## Regras

O SkyMetron não vende chaves de API. O usuário fornece suas próprias credenciais. Cada provider possui seus próprios termos, políticas, preços, limites, retenção e controles de privacidade.

## Dados enviados

Dados podem incluir prompts, contexto, trechos de vault, respostas, metadados técnicos e credenciais necessárias para autenticação. O usuário deve revisar conteúdo antes de enviar a providers externos.

## Provider local

Ollama/local pode reduzir envio externo, mas dados ainda permanecem no ambiente local do usuário e devem ser protegidos pelo usuário.
