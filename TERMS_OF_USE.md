# Termos de Uso — SkyMetron

**Versão dos termos:** 2026-06-28-v1
**Aplicável a:** SkyMetron 0.2.1-beta e superiores

## O que é o SkyMetron

SkyMetron é uma ferramenta local/desktop para organização, automação, execução assistida por IA, gerenciamento de workspace, integração com GitHub e uso de providers de IA configurados pelo próprio usuário.

## Software beta

SkyMetron 0.2.1-beta é software em desenvolvimento. Pode conter erros, instabilidades, falhas de integração e comportamentos inesperados.

## Responsabilidade do usuário

O usuário é responsável por:

- configurar suas próprias chaves de API;
- revisar ações antes de executar;
- não usar o app para fins ilegais;
- manter backups do workspace, vault e configurações;
- revisar comandos e respostas sugeridos por IA;
- não armazenar dados sensíveis sem necessidade;
- respeitar leis aplicáveis e termos de terceiros.

## Sem garantia

O SkyMetron é fornecido no estado em que se encontra, sem garantia de disponibilidade, resultado, compatibilidade com todos os ambientes, integridade de dados ou adequação a finalidade específica. O uso é por conta e risco do usuário.

## Uso de IA

Respostas geradas por IA podem estar erradas, incompletas, desatualizadas ou inseguras. Automações devem ser revisadas antes da execução. Agentes não substituem decisão humana. Ações destrutivas precisam de confirmação explícita do usuário.

## Providers externos

O app pode usar serviços externos configurados pelo usuário, incluindo GitHub, OpenAI, NVIDIA, Google, Groq, Mistral, OpenRouter, Cerebras, MiniMax e Ollama/local.

O SkyMetron não vende chaves de API. O usuário fornece suas próprias credenciais. Cada provider possui seus próprios termos de uso, preços, limites e políticas de privacidade.

## Uso proibido

É proibido usar o SkyMetron para atividades ilegais, violação de direitos, spam, phishing, coleta indevida de dados, ataques cibernéticos, exploração não autorizada de sistemas ou violação dos termos dos providers configurados.

## Privacidade e LGPD

O uso do SkyMetron também é regido pela [Política de Privacidade](PRIVACY_POLICY.md), pelo [Aviso LGPD](LGPD.md) e pelo [Data Processing](DATA_PROCESSING.md).

## Atualizações

O app pode consultar o GitHub Releases para verificar atualizações. O usuário deve revisar notas de versão antes de instalar atualizações beta.

## Alterações

Os termos podem mudar. Quando a versão dos termos ou da política de privacidade mudar, o app pode solicitar novo aceite.
